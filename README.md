## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

```bash
npm run build
npm start
```

## Deployment

This application is optimized for deployment on Vercel. The platform will automatically detect the Next.js application and build it appropriately.

## Pages & Routes

- `/` - Home page with featured products
- `/shop` - Main product catalog
- `/items` - All products listing
- `/items/[id]` - Individual product detail pages
- `/collections` - Product collections
- `/cart` - Shopping cart
- `/wishlist` - User wishlist
- `/admin` - Admin dashboard
- `/dashboard` - User dashboard
- `/login` - Authentication page
- `/about`, `/contact`, `/consultation` - Static pages

## Environment Variables

The application doesn't require any special environment variables for basic operation.
